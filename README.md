## Flappy bird neat
# How it Works
-> An Neat ai learning to play flappy bird game
-> Press "Space" to turn ON stats penel
-> Press "d" to kill all birds in current generation

# Requirements
-> python 3.+
-> pygame
-> neat-python

# How to run
'''
> git clone https://github.com/PawelHermansdorfer/flappy-bird-neat.git
> cd ./flappy-bird-neat
> python ./main.py
'''
