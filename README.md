# Flappy bird neat
## How it Works
- An Neat ai learning to play flappy bird game
- Press "Space" to toggle stats penel
- Press "d" to skip generation

## Requirements
- python 3.+
- pygame
- neat-python

## How to run
```console
> git clone https://github.com/PawelHermansdorfer/flappy-bird-neat.git
> cd ./flappy-bird-neat
> python ./main.py
```
